<div>

English Base: Divider
Spanish translation/Traducción en Español:

Definición

Definition

An all purpose, generic container for web content. There are more specific containers (header, footer, nav, section, etc) for more specific use cases. The div container is used when none of these apply. If you're grouping content for generic purposes like styling or simply logical grouping, use a div.

¿Cómo recordar?

How to Remember

Div is used to divide your web page into similarly styled sections.

    
      <div class="chapter-list">
        <ul>
          <li>
            <p>Chapter One:  Welcome to Coding</p>
          </li>
          <li>
            <p>Chapter Two:  Fundamentals of a Computer</p>
          </li>
        </ul>
      </div>
