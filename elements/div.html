<div>

English Base: Divider
Spanish translation/Traducción en Español:

Definición

Definition

An all purpose, generic container for web page content. There are different containers such as header, footer, nav, section that are for specific sections of the web page. The div container is used when none of these apply. You group content together in containers when you want to style the content in it in the same way.

¿Cómo recordar?

How to Remember

Div is used to divide your web page into similarly styled sections.

    
      <div class="chapter list">
          <ul>
              <li>
                  <p>Chapter One:  Welcome to Coding</p>
              </li>
              <li>
                  <p>Chapter Two:  Fundamentals of a Computer</p>
              </li>
          </ul>
