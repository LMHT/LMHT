<link>

English Base Word: Link
Spanish Word Equivalent:

Definición

Definition

The <link> tag establishes a connection to a resource outside of the website. It is mostly used to link a stylesheet to the document. This tag usually contains the 'href' attribute, which may tell the browser where a stylesheet is found, for instance.

¿Cómo recordar?

How to Remember

A chain is made with links, which attaches one item to another. 


      <html>
        <head>
          <link rel="stylesheet" href="style.css">
        </head>
      </html>
