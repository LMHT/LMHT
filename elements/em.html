English Base: Emphasis
Spanish translation/Traducción en Español:

Definición

Definition

Element used to stress emphasis or show priority or force on a particular section of text so it stands out from its surrounding text. Emphasis is a logical state meaning it uses physical state changes to change the text. As it's logical, it can be one physical change or multiple. Emphasis usually changes the selected text to italics. It does not need to be the same for every website. Example: You could set up emphasis for a document on quitting smoking to always assign multiple physical changes on the words "Stop Smoking Now!" where it is in italics, font-color of royal blue and always underlined.

How to Remember

Emotion also starts with em and emphasis means putting extra emotion into that section of text.
