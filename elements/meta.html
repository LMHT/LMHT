<meta>

English Base Word: Metadata
Spanish Word Equivalent:

Definición

Definition

The <meta> element is used to represent information that cannot be expressed by using <base>, <link>, <script>, <style>, and <title>. Some types of information this represents are attributes like name and content.

¿Cómo recordar?

How to Remember

Metadata is like a metalanguage because it provides a language for how to describe a language. It spells out exactly what type of language is allowed.

    
      <html>
        <meta content="author" name="John Smith">
      </html>
