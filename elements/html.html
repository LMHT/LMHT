<html>

English Base Word: Hyper Text Markup Language
Spanish Word Equivalent:

Definición

Definition

The <html> element is the root element of an HTML document. Usually it is the very first line of the document, after <!DOCTYPE> as it’s the first element, however it can be preceded by comments.

¿Cómo recordar?

How to Remember

Always remember to put this first by identifying what you are making. You’re making an HTML document so put <html> first.

    
      <html>
        <head>
          <title>Document title</title>
        </head>
      </html>
