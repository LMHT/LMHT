<ol>

English Base: Ordered List
Spanish translation/Traducción en Español:

Definición

Definition

Used to group a list of items in order with a preceding number, letter or Roman numeral. The preceding number system is indicated in the CSS file using list-style-type. The actual starting number is in the HTML file using the start attribute.

¿Cómo recordar?

How to Remember

For a list to be in order, it has to be tagged with letters or numbers to keep it in order.

    
      <ol start="10">
        <li>The Tenth Item</li>
        <li>The Eleventh Item</li>
      </ol>
