Introducción a LMHT

Introduction to LMHT

Welcome to LMHT. This is an open source code repository to help Spanish speakers translate and understand HTML tags. Originally started as a project by Team Open Brace in the YWebCA Summer of 2015 class, this project is now being developed further as an open source project by Adorable to expand its possibilities. With your help, we can turn this into an excellent learning tool for bilingual programmers.

¿Cómo utilizar LMHT?

How to Use LMHT

Begin your search by typing in the name of the tag that you want to learn more about in the search box above. It will automatically populate with available tags. Remember this is a work in progress, so we do not have all tags yet.
Once you have selected the tag you want, the window will open and show the following key pieces of information: tag name; English base; Spanish translation; audio recording of the tag's pronunciation and definition; definition in English and Spanish; helpful hints for how to remember the meaning of the tag; sample of the tag being used in code

How to Contribute(spanish)

How to Contribute to LMHT?

If you have suggestions on missing tags and adding them, you can help contribute by going to our GitHub site: https://github.com/LMHT/LMHT.github.io. Read the contributing.md and begin helping out. All help is greatly appreciated. If you're not familiar with using GitHub, you can email us with any feedback or questions at lmht@adorable.io. We will try our best to get back to you within a couple of days.
