<a>

English base word: anchor
Spanish word equivalent: ancla

¿Cómo recordar?

El ancla de HTML es como el ancla de un barco. El ancla de un barco mantiene el barco en su posición. Un ancla de HTML mantiene el posición dentro de la misma página web o el ubicación de otro página web.

How to Remember

The HTML anchor works like a boat anchor. A boat anchor holds the boat in its location. An HTML anchor holds a location within the same web page or the location of another web page.
