<a>

English base word: Anchor
Spanish word equivalent: División

¿Cómo recordar?

'div' is probablemente el más genericó elemento de HTML que hay. No tiene que tener un significado inherente o difundir cualquier estilo. Es ultilizado simplemente a "dividir" (o juntar) el contenido dentro de una pagina.

How to Remember

A "div" is probably the most generic HTML element there is! It doesn't have any inherint meaning or impart any styles. It is used simply to "divide" (or group together) content. 

<h1>This is a cool thing.</h1>
              <div class="subhead">
                   <p>stuff</p>
              </div>

<body>

English base word: Body
Spanish word equivalent: Descanso/Romper

¿Cómo recordar?

"Break" es descanso en inglés y también es el verbo para romper. En este caso, estamos rompiendo un bloque de texto, donde creamos un nuevo linea de texto.

How to remember

A "break" tag is named that way because it "breaks" a line of text. Anything after the <br> tag will be wrapped down onto the next line.

<address>
         YWebCA<br>
         123 Main St.<br>
         Madison, WI 53288
